Mini Grocery Delivery App – Backend Server

Backend server for the Mini Grocery Delivery App (Blinkit Style) developed for the OceanX Agency Kotlin Android Assignment.

This server handles:

Product APIs
User authentication
Cart management
Order handling
Product image uploads
MongoDB database operations

Node.js

MongoDB

REST API
